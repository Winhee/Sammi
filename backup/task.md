## 디자인 시스템

### 색상 팔레트
```css
--primary: #0B2447;    /* 깊은 네이비 */
--secondary: #19376D;  /* 밝은 네이비 */
--accent: #576CBC;     /* 하이라이트 블루 */
--background: #F5F7F9; /* 밝은 그레이 */
--text: #1B1B1B;      /* 진한 그레이 */
```

### 타이포그래피
- 제목: Pretendard Bold
- 본문: Pretendard Regular
- 기술 사양: Roboto Mono

### 반응형 브레이크포인트
```css
--mobile: 640px;
--tablet: 768px;
--laptop: 1024px;
--desktop: 1280px;
```
